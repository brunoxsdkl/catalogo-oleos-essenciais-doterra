# Catálogo de Óleos Essenciais dōTERRA

Catálogo bonito, clean e objetivo para vendedoras.

**34 óleos essenciais** com fotos reais dos frascos e descrições breves baseadas no **Guia de Produtos oficial dōTERRA 2024** (a partir da segunda página).

## Características
- Design elegante e profissional (cores naturais, tipografia refinada)
- Cards com imagem + nome + descrição objetiva
- Busca por nome
- Filtros por benefício: Relaxamento • Pele • Purificação
- Modal com detalhes
- Totalmente responsivo (ótimo para celular)
- Pronto para impressão

## Como usar
Abra o arquivo `index.html` em qualquer navegador.

Ou acesse a versão online após o deploy.

## Fontes
- Descrições extraídas do Guia de Produtos dōTERRA Brasil 2024
- Imagens dos óleos da sua pasta de referência

Feito com foco em simplicidade, beleza e objetividade para facilitar a apresentação para clientes.
